The "declarativeNetRequest" permission is required to replace Gizmo's JavaScript bundle with the patched version.

When a user visits app.gizmo.ai, the browser fetches Gizmo's entry bundle (entry-*.js). The extension registers a single dynamic DNR rule at install time that blocks the original bundle request matching:

  *://app.gizmo.ai/_expo/static/js/web/entry-*.js

This ensures only the patched version — which has hearts and hints unlocked — executes on the page. Without this rule, both the original and patched bundles would run in sequence, producing JavaScript errors and undefined behavior.

Only one rule is registered. The rule targets a specific URL pattern scoped to app.gizmo.ai scripts. No browsing history or request content is read; the rule operates purely on the URL match.
